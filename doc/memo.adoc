== 保存ファイル

=== 保存先

|===

=== ファイルフォーマット

[options="autowidth"]
|===
|種別 | フォーマット | 備考

| 音楽 | m4a |
|動画 | MOV |
| 写真 | HEIC/MOV | 2ファイル生成される(どちらも1ファイルで表示可能)
|iTunes経由でコピーしたもの | 任意 | 対応フォーマットに限る
|===

MOVやHEICはiso base media formatの一種

2つ作られるんなら高効率の圧縮形式選ぶ意味ないやん
